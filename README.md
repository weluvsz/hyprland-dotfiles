# how to install 

## Arch-based

### packages
#### arch-repos

>sudo pacman -Sy hyprland waybar wofi swww git ttf-jetbrains-mono-nerd

#### AUR

>paru/yay -S hellwal waypaper wofi-emoji

## dotfiles

>git clone https://github.com/weluvsz/hyprland-dotifles.git
>cd hyprland-dotfiles
>./install.sh
